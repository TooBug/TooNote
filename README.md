# TooNote

## 官网

<https://xiaotu.io>

## 介绍

- Markdown书写
- 实时预览
- 笔记分类归档
- 笔记数据内置，不需要单独保存文件
- 支持导出HTML和PDF
- 支持todo模式
- 支持微信小程序

详情请点击浏览<https://github.com/TooBug/TooNote/blob/master/src/docs/welcome.md>

## 开发

Electron + Vue 2 + Vuex 2

开发时在源码目录（src）下执行

```sh
npm start
```

即可。

## 打包

```sh
# for max osx
npm run build:osx
# for windows (both x86 & x64)
npm run build:win
```
